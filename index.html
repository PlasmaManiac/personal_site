QSip [Coming Soon]
While Quasi-Netwon (QN) methods are generally the algorithms of choice for smooth, unconstrained optimization, QN methods don't generalize easily in the sharp-edged, kinky world of non-smooth optimization. This is a modest step forward - an implementation of a L-BFGS solver made for a class of problems which prove to be extremely useful. See the examples below. Not only do we achieve up to a 50x less iterations on average (no cherry picking here), it seems to finds better local minima in non-convex problems!

Examples
LASSO, Logistic Regression, Maximum-Volume inscribed Ellipsoid, Non-Negative Matrix factorization, Dictionary Learning, Tensor Factorization, Deep Learning.
